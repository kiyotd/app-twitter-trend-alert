# README

## Description

- tweepy, Twitter API を利用して、トレンドに任意の指定した文字列があればコンソールとデスクトップへと通知を行うアプリケーションです。

## Getting Started

### Clone

```bash
git clone https://github.com/kiyotd/app-twitter-trend-alert.git
```

### Runs the app

```bash
cd app-twitter-trend-alert
pipenv install
```

```bash
pipenv run python main.py
```
